Who am I?

I'm a Solutions Engineer with a background in software development, leadership, and customer service. At my current company, I help build full-stack applications that bridge the gap between technology and real-world user needs in hiring.

What have I done?

Snake Game

A simple snake game, built off of this tutorial. This uses Vanilla Javascript, HTML Canvas, Google Fonts, and Bootstrap Icons. I added a high score function, a directional button display for mobile devices, and some styling. New updates feature better styling and wall collisions will result in a game over. Next updates will be sound effects and the ability to pause/begin gameplay with a button.

Play here!

Headache Journal

This is a project for those with chronic pain, specific to headaches and migraines. This full-stack app was made with React, JWT Auth, MongoDB Atlas, Bootstrap, and more. The next features I'm adding are a checklist feature for potential triggers, and implementing an API to pull in the weather information.

MyBubble

MyBubble is a Covid-era app that uses augmented reality to show you where your personal space bubble is, to promote proper social distancing. This was a team project for the TechTogether Boston's hackathon, in part with Women Who Code Boston, and it was so much fun to build! MyBubble took home the award for Best Community Enrichment Hack. It uses echoAR's cloud platform, and was designed by me, Zaina Shawar, Vaibhavi Khamar, and Thao Pham.

See it here!

Beets

Beets is a social media web app designed with Spotify Web API to allow users to create profiles, playlists based on vibe, and add to those playlists with the songs they're listening to in real time! This was a group project for Unit 3 of General Assembly SEI, made with Cory Spicer and Kailah Konkel. Uses JSON Web Tokens, Bootstrap, Express, MongoDB Atlas, Spotify Web API, React, and more.

Suncatcher

This project calculates sunset and sunrise times based on the user's IP address. It makes two API calls: the first retrieves the user's location, and the second fetches the corresponding times in UTC. As a beginner, I wasn't able to fully convert the UTC times to local time for this project. Built with HTML, CSS, JavaScript, and APIs, this was my very first project for the Software Engineering Immersive program at General Assembly. Although the functionality isn't fully working anymore, the visual design is still live — feel free to check it out! I plan to update and improve it in the future.

What have I done?

Snake Game

A simple snake game, built off of this tutorial. This uses Vanilla Javascript, HTML Canvas, Google Fonts, and Bootstrap Icons. I added a high score function, a directional button display for mobile devices, and some styling. New updates feature better styling and wall collisions will result in a game over. Next updates will be sound effects and the ability to pause/begin gameplay with a button.

Play here!

Headache Journal

This is a project I worked on for myself. I have chronic headaches and migraines, and my doctors have suggested keeping a journal of these events. This full-stack app was made with React, JWT Auth, MongoDB Atlas, Bootstrap, and more. The next features I'm adding are a checklist feature for potential triggers, and implementing an API to pull in the weather information.

MyBubble

MyBubble is an app that uses augmented reality to show you where your personal bubble is, to promote proper social distancing. This was a team project for the TechTogether Boston's hackathon, in part with Women Who Code Boston. MyBubble took home the award for Best Community Enrichment Hack! It uses echoAR's cloud platform, and was designed by myself, Zaina Shawar, Vaibhavi Khamar, and Thao Pham.

See it here!

Beets

Beets is a social media web app designed with Spotify Web API to allow users to create profiles, playlists based on vibe, and add to those playlists with the songs they're listening to in real time! This was a group project for Unit 3 of General Assembly SEI, made with Cory Spicer and Kailah Konkel. Uses JSON Web Tokens, Bootstrap, Express, MongoDB Atlas, Spotify Web API, React, and more.

Check it out here!

Wedding Website

A custom-designed wedding website, made for my Unit 2 General Assembly SEI project. A place for guests and friends to see information about our wedding, write a note on the digital guestbook, and view information on our registry items. Uses Google OAuth 2.0, HTML, CSS, JS, Express, MongoDB, Mongoose, and EJS.

View the website here!

Suncatcher

Sunset and sunrise time calculated by user IP address. Uses two API calls, the first to grab a user's location based on their IP address, and then the second returns the times in UTC. Utilizing HTML, CSS, Javascript, and APIs. This was my first project for General Assembly's SEI program.
